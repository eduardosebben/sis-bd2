Procedures:
 PROCEDURE `p_insere_leitor` 
 insere um leitor automaticamente na tabela de leitores, facilitando esta tarefa.

Função:
 FUNCTION `Inclui_registro_Autor`
 Insere um Autor no banco de dados da rede social de leitura, e retorna a mensagem de sucesso.

View: 
 VIEW `quantidade_amg`
 Cria uma view contando o numero de leitores existentes no na tabela de leitores. 

Trigger
 TRIGGER MINHAS_EDITORAS
 Trigger para que seja inserido varias editroas na tabela de editoras no banco social de leituras.
